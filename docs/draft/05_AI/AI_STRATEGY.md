# WEB FACTOR Enterprise Documentation
## TOM VIII — AI

### 28. AI Strategy
Wzbogacanie "Gotowego Biznesu" sztuczną inteligencją, aby zaoszczędzić czas Partnera.
*   **Agenty w tle:** SEO Agent (automatyczne generowanie meta tagów do produktów), Marketing Agent (podpowiedzi postów), Support Agent (chatbot dla klientów końcowych), Catalog Agent (automatyczne tłumaczenia i opisy z jednego zdjęcia towaru).

### 29. Workflow Engine
Silnik automatyzacji procesów (If This Then That) wewnątrz sklepu Partnera.
*   **Możliwości:** Np. "Jeśli zamówienie > 500 PLN (Condition/Trigger), to (Action) nadaj tag VIP na Customer i wyślij podziękowanie".

### 30. Marketplace
Ekosystem przyszłości.
*   SDK dla zewnętrznych programistów do tworzenia "Packages" (Theme, Capability).
*   Nadzór jakości (Moderation), model dzielenia się przychodami (Revenue).
